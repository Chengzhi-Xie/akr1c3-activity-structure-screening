# akr1c3-activity-structure-screening
A multistage computational drug discovery workflow for AKR1C3 inhibitor candidate identification, integrating ChEMBL-based ligand curation, QSAR activity prioritization, ROCS/EON 3D similarity refinement, ensemble docking with 1ZQ5 and 8RRJ, and molecular dynamics validation.
